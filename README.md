# 아임포트 결제 시스템

## 1. 개요
- 본 프로젝트는 아임포트를 이용한 결제시스템을 구현하였습니다.

## 2. 개발기간
2022.12.17 ~ 2022.12.18

## 3. 개발환경
  - `NestJS v9.1.2`
  - `React 18.2.0`
  - `MySQL v8.028`
  - `Node v16.17.0`
  - `npm v8.15.0`
  - `TypeORM v0.39`


## 4. 구현목록
- 상품 목록 생성
  - 여러 가지 생성이 가능
- 아임포트 API로 결제 구현
- 아임포트 API로 환불 구현
